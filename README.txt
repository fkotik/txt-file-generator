Важно! Для запуска проекта требуется среда разработки с поддержкой .NET 6.0

1.Что бы запустить проект откройте exe по расположению "\txt file generator\txt file generator\bin\Debug\net6.0\txt file generator.exe"

2.При вводе пути к папке где будут генерироваться файлы нужно поставить в конце знак '\' (Например "C:\Users\Admin\Desktop\")

2.Пример пути к файлу от куда будет браться список слов "C:\Users\Admin\Desktop\RUS.txt"
